# Scramble

Scrambles the order of characters inside a string randomly.

1. strings - scrambles characters.
2. numbers - scrambles digits.
3. boolean - generates random boolean.

## Options

None

## Examples

```
{ 'firstName': 'John'} => { 'firstName': 'ohJn'}
```
