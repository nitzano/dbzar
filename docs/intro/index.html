DBZar

Intro​

DBZar (Database + "stranger" in Hebrew) is a a tool for anonymizing databases with a single command, just add a connection string and anonymize away!

Great for:

  1. Anonymizing production servers for local development.
  2. General utility to manipulate existing databases easily.

⭐ Highlights​

  • Use via CLI or API
  • Supports many databases out of the box
  • Anonymize single columns and entire databases
  • Manipulate data by masking, scrambling, faking and more
  • Highly configurable
